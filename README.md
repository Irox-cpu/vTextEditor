# vTextEditor
A simple Text Editor which is written on winforms c# with useful features: Zoom, Themes, File Saving, etc.

# Features
  Dark Theme,
  Light Theme,
  Show Toolbar,
  Show Above Other Windows,
  Window Opacity,
  Font Changer
  
# Info
Created on Winforms, language is c#. Program is used: Visual Studio.
